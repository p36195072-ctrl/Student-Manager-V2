Student Manager V2

A beginner-friendly Python project that demonstrates how to manage student records using dictionaries, loops, and menu-driven programming.

📌 Description

This program allows users to:

Add new students
Search for a student
View all students
Delete student records
Exit the application

Student information is stored using nested dictionaries, making it easy to organize and manage data.
